## Project Description & Scope
This project focuses on building an end-to-end Machine Learning text classification utility designed to automatically detect and screen out automotive-related email spam (such as fake warranty expirations and predatory insurance quotes) from legitimate vehicle-related operations and communications (Ham).

### Key Features Executed:
1. **Data Cleaning:** Handled missing values, stripped whitespaces, and normalized text using Pandas.
2. **Data Visualization:** Generated a dataset distribution bar chart to visualize balance between categories.
3. **Prediction Modeling:** Implemented a Text Vectorizer and Multinomial Naive Bayes classifier via Scikit-Learn.
4. **Dashboard Creation:** Engineered a responsive native GUI desktop dashboard application layout using Tkinter for real-time phrase evaluation scans.
